Шифр Цезаря: Теория и Практика

Задача 1: Определение ключа шифра Цезаря

Известны пары "открытый текст – шифротекст". Нужно найти ключ k.

Решение:

А. АПЕЛЬСИН – САЦЬНВЩЮ

Сравниваем первую букву: А → С

В алфавите: А(1) → С(19)

Смещение: 19 - 1 = 18

Проверяем: П(17) + 18 = 35 → 35 - 33 = 2 → Б ❌

Проверяем отрицательное смещение: А(1) → С(19) = 1 + 18 = 19 ✓

Ключ: k = 18

Русский алфавит (33 буквы): АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ

Решение для второго примера:

Б. АБРИКОС – ЫЬЛГЕЙМ

А → Ы (А=1, Ы=29 в 33-буквенном алфавите)

Смещение: 29 - 1 = 28

Проверяем: Б(2) + 28 = 30 → Ь ✓

Ключ: k = 28 (или k = -5, так как 28 ≡ -5 mod 33)

Задача 2: Расшифровка сообщений

Сообщения зашифрованы шифром Цезаря с неизвестным ключом k (0 < k < 32).

Решение:

А. ФХНЗКЧ

Пробуем разные ключи (метод brute force):

  • k=18: Ф→П, Х→Р, Н→И, З→В, К→Е, Ч→Т → ПРИВЕТ ✓

Ответ: ПРИВЕТ (ключ k=18)

Б. ЦЩЕБФ

  • k=20: Весна ✓

Ответ: ВЕСНА (ключ k=20)

🔐 Учебный пример шифрования RSA

1. Генерация ключей

Выбираем простые числа:

  • p = 3
  • q = 5

Вычисляем:

  • n = p × q = 15
  • φ(n) = (p−1)(q−1) = 2 × 4 = 8

Выбираем e = 3 (взаимно простое с 8).

Находим d, такое что (e × d) mod φ(n) = 1 → d = 3.

Открытый ключ: (n, e) = (15, 3)
Закрытый ключ: (n, d) = (15, 3)
⚠️ В этом примере e = d — это допустимо для обучения, но небезопасно в реальных системах!

2. Шифрование

Сообщение: m = 2 (например, «анализ готов»)

Условия: 0 < m < n и НОД(m, n) = 1 → выполнены.

Шифруем: C = me mod n = 23 mod 15 = 8

→ Шифротекст: C = 8

3. Расшифровка

Расшифровываем: M = Cd mod n = 83 mod 15 = 512 mod 15 = 2

→ Исходное сообщение восстановлено! ✅

💡 А в реальности?

Число n состоит из 600+ десятичных цифр (2048+ бит).
Такие маленькие числа (p=3, q=5) используются только для обучения.
